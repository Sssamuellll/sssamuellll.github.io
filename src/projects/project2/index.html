GAME ENGINE

GRADE: 74%

Platform | Windows
Engine | None, C++
Project Duration | September 2024 - January 2025
Tools Used | Visual Studio, GitHub, Cmake, SDL2, OpenGL, OpenAL, GLM, Doxygen, Umbrello (UML), Adobe Photoshop
Methodologies | Entity Component System (ECS), Object Oriented Programming, Matrices
OVERVIEW

This is a small game engine with the capabilities to create more complex games. This is due to how the foundation of the engine was developed using the Entity Component System (ECS) architecture approach in conjunction with various labraries. This approach favours composition over inheritance although, Object Oriented Programming and inheritance are still utilised, they are not as prominent.
